Última atualização: 15 de janeiro de 2025

Resumo: O Entrega+ é um aplicativo de roteirização inteligente para entregas. Coletamos apenas os dados necessários para fornecer o serviço de otimização de rotas e não compartilhamos suas informações pessoais com terceiros.

1. Informações que Coletamos

1.1 Dados de Localização

  • Localização atual: Para definir o ponto de partida (depósito)
  • Endereços de entrega: Coordenadas dos pontos de entrega
  • Dados de navegação: Para otimizar rotas e fornecer direções

1.2 Dados de Uso

  • Endereços digitados ou capturados via câmera/QR code
  • Preferências de roteamento
  • Histórico de rotas otimizadas (armazenado localmente)
  • Imagens capturadas pela câmera: Apenas para OCR (reconhecimento de texto) e processamento local

1.3 Dados Técnicos

  • Informações do dispositivo (modelo, sistema operacional)
  • Logs de uso do aplicativo
  • Dados de performance e erros

2. Como Usamos suas Informações

2.1 Funcionalidades Principais

  • Otimização de rotas: Usamos suas coordenadas para calcular a melhor sequência de entregas
  • Navegação: Fornecemos direções precisas entre os pontos
  • Histórico: Mantemos um registro local das suas rotas para melhorar a experiência
  • Reconhecimento de texto (OCR): Processamos imagens capturadas pela câmera para extrair endereços automaticamente
  • Captura de QR codes: Digitalizamos códigos QR para obter informações de endereços

2.2 Melhorias do Serviço

  • Análise de padrões de uso para otimizar algoritmos
  • Correção de bugs e melhorias de performance
  • Desenvolvimento de novas funcionalidades

3. Compartilhamento de Informações

Importante: Não vendemos, alugamos ou compartilhamos suas informações pessoais com terceiros para fins comerciais.

3.1 Serviços de Terceiros

  • Google Maps: Para exibição de mapas e navegação
  • Google ML Kit: Para reconhecimento de texto em imagens
  • API de Otimização: Enviamos apenas coordenadas para cálculo de rotas

3.2 Exceções Legais

Podemos compartilhar informações apenas quando:

  • Exigido por lei ou ordem judicial
  • Necessário para proteger nossos direitos legais
  • Para prevenir fraudes ou atividades ilegais

4. Armazenamento e Segurança

4.1 Armazenamento Local

  • Dados de endereços são armazenados localmente no seu dispositivo
  • Histórico de rotas fica no seu smartphone
  • Você pode limpar esses dados a qualquer momento

4.2 Transmissão Segura

  • Todas as comunicações são criptografadas (HTTPS)
  • Dados de localização são enviados apenas quando necessário
  • Não armazenamos dados sensíveis em nossos servidores

5. Seus Direitos

5.1 Controle dos Dados

  • Acesso: Você pode ver todos os dados armazenados no app
  • Correção: Pode editar ou corrigir informações incorretas
  • Exclusão: Pode remover endereços e limpar o histórico
  • Portabilidade: Pode exportar seus dados de endereços

5.2 Permissões

  • Localização: Necessária para definir depósito e navegação
  • Câmera: Para capturar imagens de endereços (OCR), QR codes e foto de perfil. Importante: As imagens são processadas localmente no seu dispositivo e não são armazenadas em nossos servidores.
  • Armazenamento: Para salvar dados localmente

6. Uso da Câmera - Detalhes Importantes

Transparência sobre o uso da câmera: Nosso aplicativo solicita permissão de câmera apenas para funcionalidades específicas e essenciais ao serviço de otimização de rotas.

6.1 Finalidades do Uso da Câmera

  • Reconhecimento de texto (OCR): Para extrair automaticamente endereços de documentos, etiquetas ou placas
  • Captura de QR codes: Para digitalizar códigos QR que contenham informações de endereços
  • Foto de perfil: Para personalizar a conta do usuário (opcional)

6.2 Processamento e Armazenamento

  • Processamento local: Todas as imagens são processadas diretamente no seu dispositivo
  • Sem armazenamento em servidor: Não enviamos nem armazenamos suas imagens em nossos servidores
  • Dados extraídos: Apenas o texto reconhecido (endereços) é utilizado para otimização de rotas
  • Controle do usuário: Você pode negar a permissão de câmera e usar a digitação manual de endereços

6.3 Segurança e Privacidade

  • As imagens capturadas são processadas em tempo real e descartadas imediatamente após o OCR
  • Não compartilhamos imagens com terceiros
  • O processamento OCR utiliza a biblioteca Google ML Kit, que garante privacidade e segurança
  • Você pode revogar a permissão de câmera a qualquer momento nas configurações do dispositivo

7. Menores de Idade

O Entrega+ não é destinado a menores de 13 anos. Não coletamos intencionalmente informações de crianças. Se descobrirmos que coletamos dados de um menor, removeremos essas informações imediatamente.

8. Alterações na Política

Podemos atualizar esta política de privacidade periodicamente. Quando fizermos alterações significativas:

  • Notificaremos através do aplicativo
  • Atualizaremos a data de "última modificação"
  • Forneceremos um resumo das mudanças

9. Base Legal (LGPD)

No Brasil, processamos seus dados pessoais com base em:

  • Execução de contrato: Para fornecer o serviço de otimização de rotas
  • Interesse legítimo: Para melhorar nossos algoritmos e serviços
  • Consentimento: Para funcionalidades opcionais como notificações
